## Scape

Scape is an application that allows you to rank the best study spots (or the worst) at DePaul University.  
Simply select any campus (Loop or Lincoln Park). Then, select up any building you’d like. Maybe it’s a   
building nearest to you, or a building you happen to have a class in. Keep track of all of your reviews!  
Contribute to ratings by adding your own review. Don’t see a review for your favorite study spot?  
Add a new review for a spot you’d love to share! Don’t gatekeep! Upload your own pictures and rank  
study spots based on noise, business, comfort, and aesthetics. Scape was built using the SwiftUIKit  
and the XCode IDE.
